# Федерация Холода — Сайт компании по ремонту холодильного оборудования

Современный многостраничный сайт для компании «Федерация Холода», специализирующейся на профессиональном ремонте, монтаже и обслуживании холодильного оборудования.

## 🛠 Технологический стек

- **Next.js 14+** (App Router) — React-фреймворк для производства
- **TypeScript** (strict mode) — типизация для надёжности кода
- **Tailwind CSS** — utility-first CSS фреймворк
- **Framer Motion** — плавные анимации при скролле и взаимодействии
- **Lucide React** — современная библиотека иконок
- **next-seo** — SEO оптимизация через metadata API

## 📄 Структура сайта

- `/` — Главная страница с hero-блоком, преимуществами, услугами
- `/services` — Детальная страница услуг с прайс-листом
- `/about` — Страница о компании (история, команда, гарантии)
- `/contacts` — Контакты с формой заявки
- `/admin` — Админ-панель для редактирования контента

## 🚀 Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск dev-сервера

```bash
npm run dev
```

Сайт будет доступен по адресу: http://localhost:3000

### 3. Сборка для продакшена

```bash
npm run build
npm start
```

## 🎨 Дизайн

- **Цветовая палитра**: холодные оттенки (основной: #0F172A, #1E3A8A, #3B82F6, акценты: #06B6D4)
- **Типографика**: Inter (системный стек + Google Fonts)
- **Mobile-first**: корректное отображение на 320px, 768px, 1024px, 1440px+
- **Анимации**: плавные появления при скролле (Framer Motion)
- **Доступность**: семантические теги, alt-тексты, контрастность, focus-состояния

## � Админ-панель

Админ-панель доступна по адресу `/admin`.

**Демо-пароль**: `admin123`

В админ-панели можно редактировать:

- Hero-секцию (заголовок, описание, CTA)
- Преимущества компании
- Услуги и категории
- Прайс-лист (цены на услуги)
- Контактную информацию

Все изменения сохраняются в файл `data/content.json` и сразу отображаются на сайте.

## 📊 Управление контентом

Контент сайта хранится в файле `data/content.json` в формате JSON. Вы можете редактировать его напрямую или через админ-панель.

Структура файла:

```json
{
  "hero": { ... },
  "advantages": [ ... ],
  "services": [ ... ],
  "priceList": [ ... ],
  "contacts": { ... },
  "company": { ... }
}
```

## 📞 Контакты

Для изменения контактных данных:

1. Откройте админ-панель `/admin`
2. Перейдите в раздел "Контакты"
3. Измените данные и сохраните

Или отредактируйте файл `data/content.json` напрямую.

## 🔧 Настройка

Для изменения цветов отредактируйте `tailwind.config.ts` — секция `theme.extend.colors`.

Для изменения шрифтов отредактируйте `app/layout.tsx` — настройка `Inter` шрифта.
